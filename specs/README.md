# Specs — Internal Development Documents

The files in this directory are internal product and engineering specifications used during development. They are **not** user documentation.

**All personal names, locations, school names, and account numbers** appearing in these files are **fictional examples** (the "Patel family" — Raj, Priya, Arjun, Ananya) used to illustrate Artha's capabilities. They do not represent real people.

Your actual personal data lives in `config/user_profile.yaml` — which is gitignored and never committed to this repository.
